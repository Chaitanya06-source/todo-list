# todo-list
A simple, interactive todo list web application built with HTML, CSS, and JavaScript.
Add, edit, delete, and mark tasks as complete right in your browser.

Features
Add new tasks with custom text.

Mark tasks as completed (strikethrough + checkbox).

Edit existing tasks inline.

Delete tasks individually.

Persistent storage using localStorage (tasks save across sessions).

Tech Stack
Frontend: HTML5, CSS3,  JavaScript

Storage: Browser localStorage

Responsive design for desktop and mobile
